A comprehensive collection of Flutter applications demonstrating various concepts, features, and best practices in Flutter development.

## 📱 About

This repository contains 14 Flutter projects ranging from beginner to intermediate level. Each project focuses on different aspects of Flutter development, including UI design, state management, API integration, Firebase, animations, and more.

## 🏃 Getting Started

### Prerequisites
- Flutter SDK installed ([Installation Guide](https://flutter.dev/docs/get-started/install))
- Dart SDK (comes with Flutter)
- An IDE (VS Code, Android Studio, or IntelliJ IDEA)
- iOS Simulator (Mac) or Android Emulator

### Running a Project

1. Navigate to the project directory:
   ```bash
   cd project-name
   ```

2. Install dependencies:
   ```bash
   flutter pub get
   ```

3. Run the application:
   ```bash
   flutter run
   ```

### Note
Some projects (like Clima-Flutter, bitcoin-ticker-flutter, and flash-chat-flutter) require API keys or Firebase configuration. Please refer to individual project folders for specific setup instructions.
