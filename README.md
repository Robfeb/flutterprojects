## 📱 About the Repository

This repository contains **14 distinct Flutter projects**. Each project is designed to tackle specific development challenges, from basic UI layout and state management to complex integrations like REST APIs, Firebase, and real-time data streaming.

### 🛠 Tech Stack
- **Framework:** [Flutter](https://flutter.dev/) (3.0.0+ / latest stable)
- **Language:** [Dart](https://dart.dev/) (3.0.0+ / full Null Safety)
- **Backend/Services:** Firebase (Current APIs), OpenWeatherMap API, CoinAPI
- **State Management:** Provider, setState (StatefulWidgets)
- **Key Packages:** `http` 1.x, `audioplayers` 5.x, `geolocator` 11.x, `firebase_core` 3.x

## 🏃 Getting Started

### Prerequisites
- [Flutter SDK installed](https://flutter.dev/docs/get-started/install)
- IDE (VS Code, Android Studio)
- Android Emulator or iOS Simulator

### Installation & Usage
1.  **Clone the repo:**
    ```bash
    git clone https://github.com/your-username/flutterprojects.git
    ```
2.  **Navigate to a project folder:**
    ```bash
    cd project-name
    ```
3.  **Install dependencies:**
    ```bash
    flutter pub get
    ```
4.  **Run the app:**
    ```bash
    flutter run
    ```

> [!NOTE]
> Projects like **Clima**, **Bitcoin Ticker**, and **Flash Chat** require external configuration. You will need your own API keys for OpenWeatherMap and CoinAPI, and a Firebase project setup for Flash Chat.
